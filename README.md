# Beryl-Blue-Studio
 Beryl Blue Studio is a web application designed to showcase and sell handcrafted paper crafts such as greeting cards, bookmarks, and handmade albums. The website features a clean, soft color palette and a minimal UI inspired by modern craft studios. Users can browse collections, view featured products, customize their orders, add items to the cart, and proceed through a simple checkout flow.
